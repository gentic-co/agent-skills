# Gentic Agent Skills

Agent skills for [Gentic](https://gentic.co) MCP tools. Works with Claude Code, Cursor, Copilot, and [40+ other agents](https://agentskills.io).

## Install

```bash
npx skills add gentic-co/agent-skills
```

## Available skills

| Skill | Description |
|-------|-------------|
| `gentic-influencer` | Guides the influencer creator discovery workflow — brand setup, creator search, AI scoring, and export |

## What this does

Once installed, your agent automatically knows how to use Gentic's MCP tools in the right order:

1. **Brand setup** — creates your brand profile before searching
2. **Search** — finds creators on Instagram and TikTok using vector search
3. **Score** — AI-scores matches against your brand profile
4. **Review** — presents ranked results as a formatted table
5. **Export** — generates a CSV download link

## Requirements

You need the [Gentic MCP server](https://gentic.co) configured in your agent to use these skills.
